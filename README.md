# Fishing Script for QB

**[Source](https://github.com/Kuzkay/esx_AdvancedFishing)**

## Info
- Rent Fishing Boats 
- Catch 6 diffrent fish
- Fish spawn once caught
- Sell regular fish easily
- Sell exotic fish with a special card item
- few extras
